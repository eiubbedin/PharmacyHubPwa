# PharmacyHubPwa

PWA pentru gestionarea catalogului de medicamente și a comenzilor, cu backend Supabase.

Funcționalități principale:
- Nomenclator medicamente cu căutare și adăugare/ediatre.
- Comandă activă cu adăugare linii, modificare cantități și finalizare.
- Istoric comenzi cu detalii, ștergere comenzi vechi și export PDF.
- Pagina de sugestii (top medicamente comandate).

Rularea aplicației în dezvoltare:

```bash
npm install
npm run dev
```
